# NPOS 預約簡訊模板系統 — 建置提示詞（給 Claude Code）

## 角色與目標

你是一位前端工程師，要實作一個「預約簡訊模板參數化與共用模板」的單頁原型。系統有兩大模式：**預約操作頁**（套用模板、發送簡訊）與**模板設定頁**（用參數與區塊組合、維護全系統共用模板）。模板為全系統共用，但「釘選」與「最近使用」屬個人偏好。

## 技術需求

- 預設：純 HTML / CSS / JavaScript，不使用框架、不連外部 CDN，產出單一可執行 `.html` 檔。
- （若改用框架）可改為 React + Vite，邏輯與規格不變；資料層維持可抽換。
- 個人偏好（釘選、最近使用、上一次套用模板）與自訂模板用 `localStorage` 持久化，需以 try/catch 包裝並在沙箱環境退回記憶體模式（避免預覽報錯）。
- 風格：SaaS 後台。乾淨、克制；醫療/中醫調性建議玉綠主色 + 暖灰中性色，釘選用金色星號。
- RWD：桌機三欄、窄螢幕堆疊；可鍵盤聚焦；尊重 `prefers-reduced-motion`。

## 一、共用資料模型

### 系統帶入參數（發送時自動替換）
`{{會員姓名}}`、`{{門市名稱}}`、`{{預約日期}}`、`{{預約時間}}`、`{{班別}}`、`{{醫師姓名}}`、`{{門市地址}}`、`{{門市電話}}`

### 手動輸入型參數（發送前由人員填寫，非自動帶入）
`{{注意事項}}`、`{{其他備註}}`
- 凡含此兩者的模板，預覽需以**金色虛線標記**並提示「發送前手動輸入」。

### 範例（用於即時預覽替換）
```
會員姓名：林雅婷
門市名稱：京都堂 台北中山館
預約日期：2026/07/03（五）
預約時間：19:30
班別：晚診
醫師姓名：陳冠宇 醫師
門市地址：台北市中山區南京東路二段 101 號 3 樓
門市電話：02-2521-7788
```

### 模板資料結構
```
{ id, name, desc, body, creator, created, enabled, scenario:"預約簡訊" }
```
- `body` 內以 `{{參數}}` 標記，原樣儲存，套用時才替換。
- `id` 以 `b` 開頭為系統內建、`c` 開頭為自訂。

### 簡訊則數計算（中文 UCS-2）
- 字數 = 替換後純文字長度（手動參數以 `〔注意事項〕` 等佔位字數估算）。
- 則數：`len===0 → 0`；`len<=70 → 1`；否則 `ceil(len/67)`。

## 二、預約操作頁

- 左卡顯示預約資料八欄：會員姓名、門市名稱、預約日期、預約時間、班別（pill 樣式）、醫師姓名、門市地址、門市電話。
- 「選擇簡訊模板」按鈕 → 右側滑出 Drawer。
- 右卡為手機樣式簡訊預覽：套用後即時替換參數，系統參數綠底、未填/手動參數金底。
- 預覽下方顯示：目前模板名稱、字數、預估則數；含手動參數時顯示提示列。
- 「發送簡訊」按鈕（未套用模板時 disabled），點擊顯示發送 toast。

### 右側 Drawer（簡訊模板面板）
- 頂部標示「模板共用，釘選與最近使用為個人偏好」。
- 搜尋框：依名稱或用途即時過濾。
- 三區，順序為 **釘選模板 → 最近使用 → 全部模板**，各帶數量徽章；釘選與最近使用的項目不重複出現在全部模板。
- 模板卡片顯示：名稱、用途說明、（自訂模板）建立者與日期、星號釘選鈕、套用鈕（目前套用中標示）。
- 互動：點星號即時釘選/取消；點套用即時帶入預約頁並更新「最近使用」（移到最前、上限 5 筆）。
- 「新增模板」按鈕 → 快速新增表單（名稱、用途、內容 textarea、插入參數按鈕）；儲存後加入清單、自動套用、關閉表單。

### 內建範例模板（5 筆）
1. 數位客服－新客預約
2. 中醫－午診提醒
3. 中醫－晚診提醒
4. 醫美－療程預約提醒
5. 一般預約提醒

## 三、模板設定頁（三欄式建構器）

用途不是預約操作，而是建立與維護全系統共用模板。

### 左欄：可用參數面板
- 三組：**系統帶入**（綠點）、**手動輸入**（金點、虛線框、標示「發送前填」）、**常用區塊**。
- 每個參數以 chip 呈現，可：點擊插入到目前游標所在區塊、拖曳放入指定區塊、顯示說明 tooltip、標示資料來源。

### 常用區塊清單（點擊即加入編輯區）
開頭問候、預約資訊、班別資訊、門市地址、報到提醒、注意事項、其他備註、結尾文字。各區塊帶預設文字（含對應參數）。

### 中欄：模板編輯器
- 上方：模板名稱、用途說明輸入欄。
- 下方：區塊清單。每個區塊可：
  - 直接輸入文字
  - 點參數插入游標位置 / 拖參數放入
  - 拖曳握把調整順序（桌機）＋上下箭頭（手機/無障礙備援）
  - 啟用 / 停用開關（停用者不進預覽與儲存內容）
  - 刪除
- 可由「常用區塊」重複加入多個區塊，組成排列組合，例如：
  `[開頭問候] + [預約資訊] + [門市地址] + [注意事項] + [結尾文字]`

### 右欄：即時預覽
- 顯示「已套用範例資料」摘要。
- 手機氣泡：啟用區塊以 `\n\n` 串接後替換參數即時渲染。
- 統計：字數、預估簡訊則數。
- 未填參數提示：列出模板內的手動輸入欄位（注意事項 / 其他備註）。

### 情境範例（一鍵載入區塊組合）
- **一般預約**：`親愛的{{會員姓名}}您好，提醒您已預約{{門市名稱}}，日期：{{預約日期}}，時間：{{預約時間}}，如需異動請來電{{門市電話}}。`
- **中醫門診**：`親愛的{{會員姓名}}您好，提醒您已預約{{門市名稱}}{{預約日期}}{{班別}}門診，請依現場叫號順序看診。`
- **新客預約**：開頭問候 + 預約資訊（含地址）+ 報到提醒（提前 10 分鐘）+ 其他備註。

### 儲存邏輯
- 儲存欄位：模板名稱、用途說明、模板內容（由啟用區塊組合）、建立者、建立日期、啟用狀態、使用情境（預約簡訊）。
- 儲存後寫入共用清單（`localStorage`），立即出現在預約頁 Drawer 的「全部模板」，所有使用者皆可查看與套用。

## 四、互動驗收清單

- [ ] 切換「預約操作 / 模板設定」兩模式。
- [ ] Drawer 從右滑出；搜尋即時過濾；釘選即時更新並置頂；套用即時預覽並更新最近使用。
- [ ] 快速新增表單儲存後加入清單並自動套用。
- [ ] 設定頁：點擊參數插入游標位置；拖曳參數放入區塊；拖曳/上下鍵調整區塊順序；啟用/停用區塊即時反映預覽。
- [ ] 修改任一區塊內容即時更新右欄預覽、字數、則數。
- [ ] 含 `{{注意事項}}` 或 `{{其他備註}}` 時，預覽提示為發送前手動輸入欄位。
- [ ] 設定頁儲存後，模板立即出現在預約頁模板清單。
- [ ] 重新整理後仍記住釘選、最近使用、上一次模板、自訂模板（本機環境）。

## 五、可延伸（非必要，留作後續）

- 自訂模板的編輯 / 刪除 / 停用切換。
- 預約資料改為可編輯，方便測試不同會員的參數替換。
- 模板套用情境分流（中醫 / 醫美 / 新客）與權限（誰可編輯共用模板）。
- 多門市 / 多 BU 的模板範圍（全系統 vs 單門市覆寫）。
